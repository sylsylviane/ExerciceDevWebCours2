Les commandes Git

  • git -v permet de voir notre version de git
  • git config --list permet de voir les infos reliées au compte git
  • git help COMMANDE ou git config --help ouvre une fenêtre d'aide à propos de la commande
  • git init initialise un projet. La lettre U apparait à côté du fichier dans visual Code. La lettre U équivaut à Untracked. La commande Git init crée un fichier caché. C'est dans ce fichier que git enregistre les modifications. Une branche master est créée.
  • git status permet de voir le statut des fichiers. Par exemple, si des modifications ont été faites.
  • git add nomDuFichier ou git add . permet de rendre le fichier visible par git. Le U de git init devient un A pour added
  • use git rm --cached permet de revenir à U
  • git commit -m "Message" permet de faire un commit. Celui-ci vient toujours avec une description du commit
  • git log voir la liste des commits
  • git remote add origini "adresse du repository" relie le projet à l'adresse du repository
  • git push origin master pousse notre branche master sur origin
